cd C:\influxdb3-core-3.7.0-windows_amd64

./influxdb3 serve --node-id host01 --object-store file --data-dir ~/.influxdb3

| Parámetro                 | ¿Qué hace?                                        |
| ------------------------- | ------------------------------------------------- |
| `serve`                   | Inicia el motor InfluxDB 3 Core                   |
| `--node-id host01`        | Nombre único del nodo en la instancia             |
| `--object-store file`     | Guarda datos localmente en disco                  |
| `--data-dir ~/.influxdb3` | Ruta donde se guardan todos los archivos internos |
